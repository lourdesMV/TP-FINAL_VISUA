# La historia que no debemos olvidar

Proyecto de visualización de datos tipo scrollytelling sobre el ascenso del nazismo y la Segunda Guerra Mundial, desarrollado para la materia Visualización de Datos (UTDT).

El sitio recorre, capítulo por capítulo, el período que va desde el Tratado de Versalles y la crisis económica de Weimar hasta la expansión territorial alemana y el sistema de campos de concentración, combinando narrativa histórica con gráficos interactivos.

## Contenido

- **Capítulo 0** — Línea de tiempo horizontal con interacciones al pasar el mouse
- **Capítulos 1-2** — Crisis de Weimar y ascenso electoral del nazismo, con gráfico de resultados del Reichstag (1919-1933)
- **Capítulo 3** — Nombramiento de Hitler como canciller y concentración del poder
- **Capítulo 4** — Mapa animado de la expansión territorial alemana (1938 en adelante)
- **Capítulo 5** — Sistema de campos de concentración
- **Capítulo 6** — Éxodo de refugiados, con mapa y gráficos de país de destino

## Stack

HTML, CSS y JavaScript, con:
- [D3.js](https://d3js.org/) para las visualizaciones de datos y el mapa
- [GSAP / ScrollTrigger](https://gsap.com/scrolltrigger/) para las animaciones ligadas al scroll
- [Flourish](https://flourish.studio/) para gráficos embebidos (barras, dispersión)

## Fuentes de datos

Our World in Data, Maddison Project Database, Correlates of War (Inter-State War Data, alliance data).
